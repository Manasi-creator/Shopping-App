# 🛍️ Shopping App Demo

A high-fidelity E-commerce UI/UX demonstration built using **Flutter**. This project is inspired by professional design patterns to showcase a modern shopping experience with fluid transitions and clean architecture.

## ✨ Features
* **Modern Product Catalog:** Elegant grid-based display of trending items.
* **Category Filtering:** Smoothly navigate between product segments.
* **Product Details:** Immersive view with high-res images, descriptions, and size selection.
* **Dynamic Cart:** Seamless "Add to Cart" animations and state management.
* **Responsive UI:** Fully optimized for both Android and iOS screen sizes.

## 🛠️ Tech Stack
* **Framework:** [Flutter](https://flutter.dev/)
* **Language:** [Dart](https://dart.dev/)
* **Architecture:** Modular Widget Tree for high reusability.
* **UI Components:** Custom Material 3 widgets and Google Fonts.
